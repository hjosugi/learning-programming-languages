# Functional Programming Basics

Dependency-free functional programming warmup in JavaScript.

This topic is the runnable entry point before going deeper into Haskell,
Common Lisp, TypeScript type-level programming, and functional DDD.

## Run

```bash
node topics/functional-programming-basics/result.test.mjs
node topics/functional-programming-basics/result.mjs
```

## What It Teaches

- pure functions
- immutable updates
- `Option`-style absence
- `Result`-style typed failure
- composition with `pipe`
- boundary parsing before domain logic
